# Sparta_Dungeon By 이민열
구현한 과제 : 스파르타 던전 Unity버전 만들기
구현 내용
1. 메인화면 구성
 - 아이디
 - 레벨
 - 골드
 - Status 버튼 - Status 보기
 - Inventory 버튼 - Inventory 보기
 - Shop 버튼
   
2. Status 보기
 - Status버튼 클릭 - 캐릭터 정보를 보여준다.
 - 뒤로가기 버튼 클릭 - 캐릭터 정보를 다시 숨긴다.
     
3. Inventory 보기
 - 인벤토리 표시
    - ScrollView, GridLayoutGroup 사용
 - 아이템 클릭 - 장착 확인 팝업
